makeup-navigation-emitter

Set focus inside any of the dumb widgets, then use ARROW keys to update model.

This module does not do any focus management, it only models state, and triggers an event on state change (see developer console for events).

This module creates the underlying model for makeup-roving-tabindex and makeup-active-descendant. You probably don't need to use this module directly.
